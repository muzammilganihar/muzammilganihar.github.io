From My Workshop: A Showcase of Projects

The Quest for Accuracy: RAG-Based File Reader

Currently, I am spearheading a personal project to develop a file reader powered by Retrieval-Augmented Generation (RAG). I designed the system to intelligently extract and query information from various document types, like PDFs and DOCX files. By implementing a robust LangChain pipeline, my goal for this project is to significantly reduce AI hallucinations and set a new standard for data retrieval accuracy.

Vocalizer: Redefining Audio Manipulation

In early 2025, I developed "Vocalizer," a versatile audio tool using Python and Streamlit. This application empowers users with features like AI-driven vocal separation, YouTube audio downloading, and loudness normalization. I engineered the project with a modular architecture and meticulous error handling to ensure its stability and future scalability.

AI for Health: Alzheimer's Classification Model

In a significant contribution to medical technology, I worked on a model for Alzheimer's disease classification using YOLOv8. During this project in late 2024, I utilized the Roboflow platform to fine-tune the model's performance, which showcased my ability to apply advanced computer vision techniques to critical healthcare challenges.
